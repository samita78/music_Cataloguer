Steps to run the programme.
1) Import the attached sa01264_musicCataloguer.sql file on phpmyAdmin.
2) Run the program from MainCatalougerUI class.

Songs are to be added from the user's directory and the database stores the path for the audio file.

To upload and play a song in the playlistUI:
1) User must add the song from their computer, and then it can be viewed in the playlistUI where they can now play,stop,loop, edit,filter or search the song.

To add multiple songs at once- in the albumUI
1) select multiple audio files in this AlbumUIFrame and can do all the functions mentioned above. 


*The Junit test classes tests for connections and statements so the test will only pass after connecting to the database.
